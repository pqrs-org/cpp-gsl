# cpp-gsl

Utilities of `gsl`.

## Requirements

cpp-gsl depends the following classes.

- [GSL](https://github.com/microsoft/gsl)

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
